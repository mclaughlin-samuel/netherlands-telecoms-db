# Global Telecoms Database

A self-designed MySQL database modelling global telecoms operators — built as a hands-on SQL fundamentals project alongside *Learning SQL* by Alan Beaulieu.

## Database

MySQL, connected via DBeaver.

## Project goals

- Design and build a relational database from scratch
- Practice CREATE TABLE, INSERT, SELECT, JOIN, and aggregations on real-world domain data
- Model entities including operators, markets, networks, and subscribers
